#来自封面们的封面

某一天用iTunes听歌时突发奇想，想利用专辑封面做一个可视化效果。

预期基本功能：

1. 能够识别iTunes正在播放的曲目
2. 利用所有专辑的封面拼出当前播放曲目的专辑封面

预期高级功能：

1. 根据节奏产生动画效果
2. 点击可视化效果中某一专辑封面，切换到该专辑中的随机曲目
3. 嵌入iTunes中
